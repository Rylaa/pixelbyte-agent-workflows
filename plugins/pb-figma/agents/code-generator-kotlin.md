# Kotlin Compose Generator (Coming Soon)

This agent is planned for v1.2.0.

## Planned Features

- Jetpack Compose UI components with @Composable functions
- Material Design 3 theming (MaterialTheme, colorScheme)
- Kotlin best practices and idioms (data classes, sealed classes)
- State management with remember/mutableStateOf/derivedStateOf
- Accessibility with semantics modifiers (contentDescription, role)
- Dark mode support with isSystemInDarkTheme()
- Preview functions for design verification
- Design tokens as Kotlin objects

## What Works Now

Use `code-generator-react` for immediate Figma-to-code conversion.
React components can be converted to Kotlin Compose manually.
